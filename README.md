# Stock Price Predictor

> LSTM neural network for stock price prediction with technical indicators (RSI, MA, Bollinger Bands)

## Overview
This project is part of a comprehensive AI/ML portfolio covering time series forecasting and finance.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/stock-price-predictor.git
cd stock-price-predictor
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: PyTorch, numpy, pandas, scikit-learn

## Usage
```bash
python main.py
```
Run with `--help` for full options.
